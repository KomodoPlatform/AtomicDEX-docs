# AtomicDEX RPC Methods

Legacy methods are in the process of a migration to AtomicDEX API v2.0 (and will also remain backwards compatible). During development, some newer AtomicDEX API methods are not available in the latest release.

To test the methods in v2.0 (Dev), you will need to build the AtomicDEX API from the dev branch.
